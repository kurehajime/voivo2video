# 実装計画

## フェイズ１

実証実験として、vvprojファイルを読み込み、字幕動画を生成する機能を実装する。

## フェイズ２

話者ごとに字幕の文字色や縁取り色を上書きできるようにする。

## フェイズ3

立ち絵を表示する機能を実装する。
- すべてを合成した動画
- 立ち絵のみの動画 × キャラクター数
- 字幕のみの動画
というように別々に出力できるようにする。

立ち絵画像では、話者IDごとに画像を指定できるようにする。
画像はデフォルト状態と発話状態の2つを指定して、voicevoxの発話状態で自動で切り替わるようにする。

## フェイズ4
字幕のCSSに対応する。
configファイルでCSSファイルのパスを指定できるようにする。


## フェイズ5
画像の指定方法を変えたい。
文字列で配列を受け取るようにしたい

imagePath→ normalImages
activeImagePath→activeImages

加えてフレーム数の指定方法も変える。

normalFrames=[20,5]
activeImages=[8,8]

このように指定すると、normalは1枚目を20フレーム、2枚目を5フレーム表示するのを繰り返す
